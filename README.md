# VUE3-CMS-TS-PINIA

## 后端地址
<a href="https://github.com/LeoKun1231/nest-cms-api">后端仓库地址，采用nest框架</a>

## 介绍

**VUE3**+**Pinia**+**Pnpm**+**Ts**+**tailwindcss**+**ElementPlus**+**VueRouter**+**VueUse**+**axios**

## 使用步骤

```
pnpm install
pnpm lint
pnpm dev
```

## 上传代码注意

该模板使用了 husky+commitizen 管里提交代码，所以，你应该按如下步骤进行提交

```
git add .
pnpm commit
git push origin main
```
